# Feature Development Workflow

## 1. Know the business model

## 2. Know the MUI component

## 3. Test the MUI component in the web and filter the sub-components that are not needed

## 4. Customize the MUI component to match what is needed and add static data that matches the business model

## 5. Give the MUI component to AI to get the classNames and the relative CSS file

## 6. Add the MUI component to the project

## 7. Test the MUI component separately

## 8. Separate the UI component into the UI layer

## 9. Test

## 10. Create a temporary helper class for the current feature to separate any unrelated UI code

## 11. Separate the logic and model into the helper class

## 12. Test

## 13. Separate the logic code to the logic layer (model)

## 14. Test

## 15. Separate the logic (service) + data repo + create the API service function

## 16. Test

## 17. Create a responsive UI

## 18. Optimise the code
